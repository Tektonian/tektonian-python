# Shadow Hand E3M5 Simulation Assets

## Overview

This package contains robot assets for the [Shadow Dexterous Hand E3M5](https://www.shadowrobot.com/dexterous-hand-series/), specifically the left-handed version, developed by [Shadow Robot Company](https://www.shadowrobot.com/).

The subfolders contain:

- **usd_structured**: Structured USD layer files, converted from MuJoCo Menagerie MJCF

## Sources

### USD (Structured)

The structured USD model was created with the [mujoco-usd-converter](https://github.com/newton-physics/mujoco-usd-converter), converted from the MuJoCo Menagerie MJCF source. For details, see the [usd_structured README](usd_structured/README.md).
